# rtmpose_tensorrt

## Description

This repository is use the TensorRT to deploy RTMDet and RTMPose. Your computer should have these components:

- NVIDIA GPU
- CUDA
- cudnn
- TensorRT 8.x
- OPENCV
- VS2019
